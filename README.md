# Voice Assistant 🎙️

A Python-based Voice Assistant that can perform various tasks using voice commands and text-to-speech technology.

## Features

* Voice Command Recognition
* Text-to-Speech Responses
* Open Applications
* Search the Web
* Tell Date and Time
* Play Music
* Execute Basic System Tasks
* User-Friendly Interface

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* pyautogui
* os
* webbrowser

## Installation

1. Clone the repository:

```bash
git clone https://github.com/abhimaurya001/Voiceassistant.git
```

2. Move to the project directory:

```bash
cd Voiceassistant
```

3. Install required packages:

```bash
pip install -r requirements.txt
```

## Usage

Run the main Python file:

```bash
python main.py
```

After starting the program, speak your command and the assistant will respond accordingly.

## Example Commands

* Open Chrome
* Search Python tutorials
* What is the time?
* Open YouTube
* Play music
